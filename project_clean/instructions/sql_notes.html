How to Create a Stored Procedure in MySQL

Overview: A stored procedure is a saved block of SQL statements that can be reused multiple times. It is stored in the database and executed with a single call command.

Steps to Create a Stored Procedure

  1. Change the Delimiter: Before creating a procedure, set a different delimiter (such as //) because the procedure body will contain semicolons (;). 
    DELIMITER //
  2. Define the Stored Procedure: Use the CREATE PROCEDURE statement followed by the procedure name, optional parameters, and SQL block inside BEGIN and END. 
            CREATE PROCEDURE ProcedureName()
        BEGIN
            -- SQL statements go here
        END //
  3. Reset the Delimiter: After the procedure is created, set the delimiter back to the default semicolon. 
    DELIMITER ;

Example

Hereâ€™s how to create a procedure to select all clients from the Clients table:

      DELIMITER //
      
      CREATE PROCEDURE GetAllClients()
      BEGIN
        SELECT * FROM Clients;
      END //
      
      DELIMITER ;

Calling a Stored Procedure

After creation, you can call the procedure like this:

CALL GetAllClients();

Tips

  • Always set and reset the delimiter properly.
  • Procedure names should be unique within the database.
  • Procedures can include multiple SQL statements, control logic, and variables.

Summary

  • DELIMITER //
  • CREATE PROCEDURE ... BEGIN ... END
  • DELIMITER ;
  • CALL ProcedureName();

Notes: Creating a Django Project and App (macOS)

Important: On macOS, all libraries (including Django) must be installed inside the virtual environment (venv). Nothing is installed globally on the system.

  1. Install Python
    • Ensure Python 3 is installed: python3 --version
    • If not installed, use Homebrew: brew install python
  2. Create and Activate Virtual Environment
    • Create virtual environment: python3 -m venv env
    • Activate environment: source env/bin/activate
  3. Install Django Inside Virtual Environment
    • Once activated, install Django locally inside venv: pip install django
  4. Create Django Project
    • Run: django-admin startproject project_name
    • Navigate into project folder: cd project_name
  5. Create Django App
    • Run inside project folder: python manage.py startapp app_name
  6. Register App
    • Open project_name/settings.py
    • Add your app to INSTALLED_APPS list:
      'app_name',
  7. Run Development Server
    • Start server: python manage.py runserver
    • Visit http://127.0.0.1:8000 in your browser.
